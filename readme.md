# Pinterest

A Framer plugin for browsing and inserting Pinterest pins directly into your Framer projects.

## Features

- 🔍 Search for pins by keyword
- 🖼️ Infinite scroll through results
- 🎲 Get a random pin suggestion
- 🎨 Masonry layout with responsive columns
- ♿ Full accessibility support
